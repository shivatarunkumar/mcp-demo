# mcp-demo

A demonstration project for the [Model Context Protocol (MCP)](https://modelcontextprotocol.io/).

## Overview

MCP (Model Context Protocol) is an open protocol that enables LLMs to connect with external data sources, tools, and services in a standardized way.

## Getting Started

### Prerequisites

- Node.js 18+ or Python 3.10+
- An MCP-compatible client (e.g., Claude Desktop, Claude Code)

### Installation

```bash
# Clone the repository
git clone <repo-url>
cd mcp-demo

# Install dependencies
npm install
```

### Running the Server

```bash
npm start
```
